About

This is a program to see if your keys are working. Simply press your keyboard to see if input registers for that key.

If the corresponding key turns green, it works. If it doesn't, that key may not work.

Notes: 1. To test numeric keypad, turn on NumLock.
2. Some keys, such as PrintScreen, are weird and do not register in the program sometimes, if at all.
